##第三类考察形态
二叉搜索树 Binary Search Tree

####BST基本性质
- 从定义出发
    - 左子树都比根结点小
    - 右子树都不小于根结点
    
- 从效果出发
    - 中序遍历 in-order traveral 是“**不下降**”序列
    
- 性质
    - 如果一个二叉树的中序遍历不是“不下降”序列，则一定不是BST
    - 如果一个二叉树的中序遍历是不下降序列，也未必是BST
        - 比如下面这棵树不是BST，但是它的中序遍历是不下降序列
        
        ```
            1
           / \
          1   1
        
        ```

####二叉树的高度 h = (logN, N)
